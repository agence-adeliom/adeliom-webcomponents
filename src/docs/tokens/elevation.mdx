# Elevation Tokens

Elevation tokens are used to give elements the appearance of being raised off the page. Use them with the `box-shadow` property. These are especially useful for menus, popovers, and dialogs.

| Token                  | Example                                                                           |
| ---------------------- | --------------------------------------------------------------------------------- |
| `--awc-shadow-x-small` | <div class="elevation-demo" style={{boxShadow: 'var(--awc-shadow-x-small)' }}></div> |
| `--awc-shadow-small`   | <div class="elevation-demo" style={{boxShadow: 'var(--awc-shadow-small)' }}></div>   |
| `--awc-shadow-medium`  | <div class="elevation-demo" style={{boxShadow: 'var(--awc-shadow-medium)' }}></div>  |
| `--awc-shadow-large`   | <div class="elevation-demo" style={{boxShadow: 'var(--awc-shadow-large)' }}></div>   |
| `--awc-shadow-x-large` | <div class="elevation-demo" style={{boxShadow: 'var(--awc-shadow-x-large)' }}></div> |
